spa-loader-content-html

Plugin vanilla JavaScript para SPAs — v3

spa-loader-content-html es un plugin vanilla JavaScript diseñado para crear aplicaciones de una sola página (SPA) sin frameworks. Carga fragmentos HTML dinámicamente mediante fetch(), inyectándolos en contenedores del DOM definidos por rutas configuradas en un manifiesto ligero.

La versión 3 introduce lazy loading de módulos de ruta mediante import() dinámico, soporte nativo para la View Transitions API, reescritura automática de URLs en HTML inyectado, caché de módulos por ruta y un sistema de eventos personalizados para desacoplar el loader visual de la lógica de navegación.

Características Principales

🚀

Lazy Loading de Rutas

Los módulos de ruta se importan dinámicamente con import() solo cuando el usuario navega a esa ruta. Un manifiesto ligero (routeManifest) registra cada ruta con id, path y file, sin importar módulos en el arranque.

🎬

View Transitions API

Cuando el navegador soporta document.startViewTransition(), el plugin envuelve automáticamente el renderizado de la nueva ruta en una transición nativa, logrando animaciones fluidas entre páginas sin configuración adicional.

📋

Route Manifest

El routeManifest es un array de entradas { id, path, file }. El plugin resuelve la ruta activa comparando window.location.pathname normalizado con cada path, y carga el módulo correspondiente bajo demanda.

🔃

Inyección de HTML por Componente

Cada ruta define un objeto components donde la clave es el id del contenedor en el DOM y el valor es la URL del fragmento HTML. El plugin hace fetch() de cada fragmento y lo inyecta en su contenedor de forma secuencial.

🎨

CSS y JS por Ruta

Cada ruta puede declarar sus propios styles (array de { href }) y scripts (array de { src, isModule? }). El plugin los inyecta dinámicamente en el <head> al navegar a esa ruta.

⚡

Caché y Eventos Personalizados

Los módulos de ruta se cachean en un Map interno para evitar importaciones redundantes. El plugin emite eventos spa:route-loaded, spa:first-route-loaded y spa:route-load-error para desacoplar el loader visual de la navegación SPA.
